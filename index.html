Genome Editing Software Guide

Welcome to Genome Editing Software Guide

We are collecting information on software for Genome Editing.
Our collection is over 200 tools (549 tools at 2025/2/5 UPDATE). So you can find good tools supporting your research.

If there is a comment, I would be happy if you contact us:
Administrator: Kazuki Nakamae, Ph.D. : e-mail: kazukinakamae_at_mark_gmail.com
bonohu lab, Genome Editing Innovation Center, Hiroshima University, Japan

Citation info:
We described this webpage in the following review:
Nakamae, K. and Bono, H. Genome editing and bioinformatics. Gene and Genome Editing, 100018 (2022).
